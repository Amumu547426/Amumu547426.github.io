简单选择排序

执行n-1趟，每趟从剩余的未排序序列中选取最小元素，交换到正确位置。

时间复杂度$O(n^2)$，不稳定，关键字比较次数与待排序序列元素无关，总是要比较$O(n^2)$次。

void SelectSort(int A[], int n)
{
    int i,j;
    for(i=0; i<n-1; i++)
    {
        int min = i;
        for(j=i+1; j<n; j++)
        {
            // if(A[j] < A[min]) 
                min = j;
        }
        if(min != i)
        {
            int temp = A[i];
            A[i] = A[min];
            A[min] = temp;
        }
        printA(A,11);
    }
}

堆排序

建立大根堆

A[0]不放元素，A[1]表示堆顶，将一维数组视为完全二叉树，A[i]的左右孩子为A[2*i]，A[2*i+1]，建堆时，从最后一个非叶节点开始调整，参数len表示堆中元素个数。

void BuildMaxHeap(int A[], int len)
{
    int i;
    for(i=len/2; i>=1; i--)
    {
        HeapAdjust(A, i, len);
    }
}

调整大根堆

调整以A[k]为根节点的子树，需要满足堆的性质。

void HeapAdjust(int A[], int k, int len)
{
    A[0] = A[k];
    int i;
    for(i=2*k; i<=len; i*=2)
    {
        //找出k节点的key最大的孩子节点i
        if(i<len && A[i]<A[i+1])
            i++;
        /* 若满足大顶堆，即k节点key值大于孩子节点i的key，则退出；
        否则将i节点的值赋给双亲节点k，并去调整以i为根节点的子树，
        子树根节点的值始终为A[0]  */
        if(A[0] >= A[i])
            break;
        else{
            A[k] = A[i];
            k = i;
        }
    }
    //最后的k值即为最初调整的子树根节点的位置
    A[k] = A[0];
}

堆排序

堆排序，首先建立大根堆。

执行n-1次，每次将当前堆顶与堆最后一个元素交换，然后堆大小-1，调整堆。这样每次取出堆顶即最大元素，放在了数组末尾，然后调整堆，使A[1]为剩余元素的关键字最大元素。

调整时间与树高有关，时间复杂度$(nlog_2n)$，空间复杂度$O(1)$，不稳定

void HeapSort(int A[], int len)
{
    int i;
    BuildMaxHeap(A, len);
    for(i=len; i>1; i--)
    {
        int temp = A[i];
        A[i] = A[1];
        A[1] = temp;
        HeapAdjust(A, 1, i-1);
    }
}
